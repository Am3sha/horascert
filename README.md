# ISO Certification Website

A professional, corporate website for an ISO certification and quality systems organization. Built with React and pure CSS.

## Features

- **Professional Design**: Clean, corporate aesthetic with trust-building elements
- **Responsive Layout**: Fully responsive design for all devices
- **Multiple Pages**: Home, About, Services, Clients, Certificate Validation, Application, and Contact
- **Hero Slider**: Auto-rotating hero section with manual controls
- **Service Showcase**: Detailed service pages for each ISO standard
- **Contact Forms**: Validated contact and application forms
- **Certificate Validation**: Certificate verification system
- **Smooth Animations**: Professional transitions and hover effects

## Technology Stack

- React (JavaScript)
- React Router
- Pure CSS (no frameworks)
- Responsive Design

## Getting Started

### Installation

```bash
npm install
```

### Running the Development Server

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Building for Production

```bash
npm run build
```
